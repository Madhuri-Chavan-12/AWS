# AWS Route 53

Amazon Route 53 is a highly available and scalable DNS (Domain Name System) web service.  
Provides DNS as a service   

 **It is used to:**

 - Register domains

- Route internet traffic to AWS resources

- Perform health checks and DNS-level failover


**A human-readable name for an IP address.**  

---

### Hosted Zone

- A container for DNS records of a domain.

- Types:

1. Public Hosted Zone → Internet-facing domains

2. Private Hosted Zone → Internal domains (VPC only)
